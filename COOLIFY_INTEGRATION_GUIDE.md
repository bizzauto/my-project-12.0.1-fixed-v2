1. Go to Coolify Dashboard → **Projects** → **New Project**
2. Name: `bizzauto-integrated`
3. Add a **New Resource** → **Docker Compose**

### Step 2: Use the Updated `docker-compose.prod.yml`
```yaml
# Copy the entire docker-compose.prod.yml from this project into Coolify
```

### Step 3: Set Environment Variables in Coolify
In the Coolify project, add these env vars under **Environment Variables**:

| Category | Variable | Value | Source |
|----------|----------|-------|--------|
| **Database** | `DATABASE_URL` | `postgresql://postgres:xxx@supabase-host:5432/postgres` | From Supabase project settings |
| **Redis** | `REDIS_URL` | `redis://redis:6379` | Auto (internal Docker) |
| **JWT** | `JWT_SECRET` | `openssl rand -hex 32` | Generate in terminal |
| **JWT** | `JWT_REFRESH_SECRET` | `openssl rand -hex 32` | Generate in terminal |
| **JWT** | `ENCRYPTION_KEY` | `openssl rand -hex 32` | Generate in terminal |
| **App URL** | `FRONTEND_URL` | `https://bizzautoai.com` | Your domain |
| **App URL** | `API_URL` | `https://bizzautoai.com` | Your domain |
| **Evolution** | `EVOLUTION_API_URL` | `https://evolution.your-vps.com` | Evolution API URL from Coolify |
| **Evolution** | `EVOLUTION_API_KEY` | From Evolution API settings | Generated by Evolution API |
| **n8n** | `N8N_URL` | `http://n8n:5678` | Internal Docker DNS |
| **n8n** | `N8N_PASSWORD` | Secure password | Choose a strong password |
| **n8n** | `N8N_ENCRYPTION_KEY` | `openssl rand -hex 32` | Generate in terminal |
| **n8n** | `N8N_USER_MANAGEMENT_JWT_SECRET` | `openssl rand -hex 32` | Generate in terminal |
| **n8n** | `N8N_WEBHOOK_URL` | `https://bizzautoai.com` | Your domain (public webhook) |

## 🔗 Cross-Service Communication Matrix

### Who talks to whom?

| From → To | How | Protocol | URL/Endpoint |
|-----------|-----|----------|-------------|
| **App → Supabase** | Prisma ORM | PostgreSQL | `DATABASE_URL` env var |
| **App → Evolution API** | HTTP REST | HTTPS | `EVOLUTION_API_URL` env var |
| **App → n8n** | HTTP REST | HTTPS | `N8N_URL` env var → `/webhook/{id}` |
| **Evolution → App** | Webhook callback | HTTPS POST | `https://bizzautoai.com/api/evolution/webhook/{businessId}` |
| **n8n → App** | HTTP API call | HTTPS | `https://bizzautoai.com/api/...` |
| **n8n → Supabase** | Direct DB query | PostgreSQL | Configure in n8n UI with DATABASE_URL |

### Port Mapping (if deployed without reverse proxy)

| Service | Internal Port | External Port |
|---------|--------------|---------------|
| BizzAuto App | `4000` | `443` (via Coolify SSL) |
| n8n | `5678` | `443` (via Coolify SSL) |
| Evolution API | `8080` | `443` (via Coolify SSL) |
| Supabase | `5432` | Internal only |

## 🚀 Step-by-Step Wiring

### Phase 1: Database Connection
```
Coolify → Supabase → Get DATABASE_URL → Set in App env vars → Push schema
```

1. In Coolify, open your Supabase resource
2. Go to **Credentials** tab → copy **Database URL**
3. Open your App resource → **Environment Variables**
4. Paste it as `DATABASE_URL`
5. Deploy the App once (it will crash initially — that's expected)
6. **Sync Prisma schema to Supabase**:

   Open a terminal and run these commands with the database connection:
   ```bash
   # Set the DATABASE_URL (use your actual Supabase URL)
   export DATABASE_URL="postgresql://postgres:YOUR_PASSWORD@YOUR_SUPABASE_HOST:5432/postgres"

   # Install deps and generate Prisma client
   npm install
   npx prisma generate

   # Push the schema directly to Supabase (creates all tables)
   npx prisma db push

   # Seed initial data (optional — creates admin user, default business, etc.)
   npx prisma db seed
   ```

   > **Note**: You can also run `npx prisma db push` as a one-time command in Coolify's terminal.

7. Redeploy the App — it should now start successfully with all database tables created.

### Phase 2: Evolution API → App
```
Coolify → Evolution API → Set API key → Set webhook URL → App receives WhatsApp messages
```

1. Deploy Evolution API in Coolify (if not already)
2. Go to Evolution API resource → **Logs** → find the auto-generated API key
3. Note the API key value
4. In App's env vars, set:
   - `EVOLUTION_API_URL=https://evolution-api.your-vps.com`
   - `EVOLUTION_API_KEY=your-api-key`
5. In Evolution API's env vars, set:
   - `WEBHOOK_URL=https://bizzautoai.com/api/evolution/webhook`
6. Redeploy both services

### Phase 3: App → n8n
```
App triggers n8n workflows → n8n calls App APIs → Full automation
```

1. Deploy n8n in Coolify (if not already)
2. Access n8n at `https://n8n.your-vps.com`
3. Login: `admin` / your `N8N_PASSWORD`
4. In App's env vars, set:
   - `N8N_URL=https://n8n.your-vps.com`
   - `N8N_PASSWORD=your-password`

### Phase 4: Connect WhatsApp (Evolution API)
```
User opens App → configures Evolution API → scans QR → WhatsApp connected
```

1. Login to BizzAuto App at `https://bizzautoai.com`
2. Go to **Settings → WhatsApp → Evolution API**
3. Enter:
   - **Server URL**: `https://evolution-api.your-vps.com`
   - **API Key**: Your Evolution API key
4. Click **Connect** → QR code appears
5. Scan with WhatsApp → WhatsApp connected ✅

### Phase 5: Import n8n Workflows
```
Project has 3 pre-built workflows → Import them into n8n → Automate
```

1. Open n8n at `https://n8n.your-vps.com`
2. Login: `admin` / your password
3. For each workflow in `n8n/workflows.json`:
   - Click **Workflows → Add from JSON**
   - Paste the workflow JSON
   - Update any URLs to use your production domain
   - Activate the workflow

## 📂 Pre-Built n8n Workflows

The project includes 3 ready-to-use workflows in `n8n/workflows.json`:

### 1. WhatsApp Auto-Reply with AI
- **Trigger**: Incoming WhatsApp message webhook
- **Flow**: Receive → Check type → Generate AI reply → Send reply
- **Uses**: OpenRouter AI via app's API
- **Endpoint**: In n8n set this to receive webhooks from the App

### 2. Lead Auto-Capture from IndiaMART
- **Trigger**: IndiaMART lead webhook
- **Flow**: Receive lead → Save to CRM (via App API) → Send welcome WhatsApp
- **Endpoint**: `POST https://bizzautoai.com/api/leads/indiamart/{businessId}`

### 3. Google Review Auto-Reply
- **Trigger**: New review webhook
- **Flow**: Receive review → Check rating → AI generate reply → Post reply
- **Endpoint**: `PUT https://bizzautoai.com/api/reviews/{reviewId}/reply`

## 🔍 Troubleshooting

### "Evolution API not configured"
- Check `EVOLUTION_API_URL` and `EVOLUTION_API_KEY` are set in App env vars
- Verify Evolution API is running: `curl https://evolution-api.your-vps.com`
- Check Coolify logs for both services

### "n8n workflow not triggering"
- Check n8n is running: `curl https://n8n.your-vps.com/healthz`
- Verify webhook URLs are correct in n8n workflow
- Check n8n execution logs in n8n UI

### "Database connection failed"
- Verify Supabase credentials in `DATABASE_URL`
- Supabase might need to allow external connections
- Check if Supabase is running: check Coolify resource status

### "CORS errors in browser"
- Ensure `CORS_ORIGIN` and `FRONTEND_URL` match your domain exactly
- If using Coolify proxy, ensure it's forwarding headers correctly
