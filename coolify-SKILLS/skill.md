# BIZZ CRM – AI Agent Skill Specification File

## Role Title

**Senior Full-Stack SaaS Production Expert (AI Agent)**

---

## Primary Objective

Audit, debug, optimize, secure, and stabilize the BIZZ CRM project hosted on a VPS using Coolify, Docker, Next.js, Node.js, and Supabase.

The AI Agent must ensure the application becomes fully production-ready, scalable, secure, and high-performance.

---

## Immediate Scope of Work (Technical Tasks)

- **Code Audit & Debugging**: Existing code madhle bugs shodhne ani te "Production-Ready" karne (e.g., error handling and security checks).
- **VPS Optimization**: Coolify madhe server chi health check karne ani "Route not found" errors resolve karne.
- **Database Management**: PostgreSQL queries optimize karne ani bizzauto database jo saddhya down dakhavtoye, to fix karne.
- **Reverse Proxy & SSL**: Traefik/Nginx configuration check karne jyamule `bizzautoai.com` var traffic nit direct hoil.
- **Security & Scalability**: `.env` files secure karne ani VPS chi RAM/CPU usage check karun application scale karne.
